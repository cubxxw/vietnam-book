## 关于这本书

### 三层叙事结构

```
街头观察  →  系统思考  →  认知跃迁
看到了什么      为什么是这样      对我意味着什么
```

### 核心主题

- **悬浮感** — 站在两个系统之间，什么都看得清，什么都抓不住
- **Tình Cảm** — 越南人的存在方式：不因利益，因连接
- **在场性** — AI 时代里，「真实发生过」这件事越来越重要
- **脆弱性作为入场券** — 不完美才能被另一个生命的不圆满所触动
